# OOP Python Interview Questions

## 1. What is Object-Oriented Programming?
Object-Oriented Programming (OOP) is a programming paradigm that uses objects and their interactions to design applications.

## 2. What are the four main principles of OOP?
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## 3. What is a class?
A class is a blueprint for creating objects, providing initial values for state and implementations of behavior.

## 4. What is an object?
An object is an instance of a class. It contains data and methods that operate on the data.

## 5. What is inheritance?
Inheritance is a mechanism that allows one class to inherit attributes and methods from another class.

## 6. What is polymorphism?
Polymorphism allows methods to do different things based on the object it is acting upon, even if they share the same name.

## 7. What is encapsulation?
Encapsulation is the bundling of data and methods that operate on that data within one unit, typically a class.

## 8. What is abstraction?
Abstraction is the concept of hiding the complex implementation details and exposing only the essentials.

## 9. What are design patterns in OOP?
Design patterns are standard solutions to common problems in software design, providing a way to structure code effectively.

## 10. Can you explain the Singleton pattern?
The Singleton pattern ensures that a class has only one instance and provides a global point of access to it.
